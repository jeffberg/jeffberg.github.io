A hands-on practice environment for learning data analysis with Python.

Click the badge below to open the notebook directly in Google Colab:

## What's Covered

| Section | Topics |
|---------|--------|
| Part 1: NumPy | Arrays, math operations, slicing, random numbers |
| Part 2: Pandas | DataFrames, filtering, groupby, pivot tables, missing data |
| Part 3: Visualization | Bar, line, scatter, heatmap, histogram, box plots |
| Part 4: EDA | Correlation analysis, dashboards, business questions |
| Bonus | Load your own dataset or public datasets |
